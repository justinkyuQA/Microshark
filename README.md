MicroShark

🦈 A tiny network traffic learning tool.

MicroShark is a mobile-first Progressive Web App (PWA) designed to help users understand common network traffic concepts through simple visual explanations.

Rather than overwhelming users with thousands of packets and protocol fields, MicroShark focuses on answering one question:

What is happening on my network?

---

Features

- Mobile-first design
- Progressive Web App (PWA)
- Installable on supported devices
- Simple dark interface
- Protocol overview cards
- Beginner-friendly explanations

---

Included Protocols

HTTPS

Secure website traffic.

Used by most modern websites to encrypt communications.

DNS

Domain Name System lookups.

Converts names like:

google.com
github.com
youtube.com

into IP addresses.

ARP

Address Resolution Protocol.

Allows devices on a local network to discover each other.

---

Philosophy

MicroShark is part of the Micro tool family.

The goal is to create small, focused tools that teach real networking concepts without unnecessary complexity.

Every Micro tool should answer one simple question.

MicroMap:

«Who is on my network?»

MicroShark:

«What is happening on my network?»

MicroWall:

«What should I trust?»

Installation

Clone the repository:

git clone https://github.com/justinkyuQA/Microshark.git
cd Microshark

Run a local web server:

python -m http.server 8000

Open:

http://localhost:8000
